03 — EmbeddingsMeaning as Numbers

Each token is mapped to a point in a 50-dimensional vector space — words with similar meanings end up nearby.

  • Count which words appear near each other
  • Weight by statistical surprise (PPMI)
  • Compress 1000D → 50D via truncated SVD
  • No human labels. Pure statistics.
The model discovers that "king" and "queen" are related — nobody told it.
