08 — AI AgentsFrom Model to Agent

  • An agent is a while loop around the transformer — the model itself hasn't changed
  • Tools— model emits structured tokens {"tool":"search","args":{...}}→ your code executes → result injected back as tokens
  • Memory— context window = short-term  ·  vector DB / files retrieved via RAG = long-term
  • Planning— ReAct (Reasoning and Action) pattern: Reason about the goal, Act with a tool, Observe the result, repeat
  • Reasoning models (o1, o3, Claude thinking) — generate hidden Chain-of-Thought (CoT) tokens before the final answer; better at multi-step problems, costs more tokens
The model never “does” anything. Your code does — based on what the model outputs. Intelligence emerges from the loop.

08 — DemoThe Agent Loop


                  async function agent(goal) {
                    let ctx = [{ role: 'user', content: goal }];

                    while (true) {
                      const response = await llm(ctx);
                      const action   = parseAction(response);

                      if (action.type === 'answer')
                        return action.text;

                      // run the tool your code controls
                      const obs = await tools[action.tool](action.args);

                      ctx.push({ role: 'assistant', content: response });
                      ctx.push({ role: 'tool',      content: obs });
                      // loop → llm sees the result and reasons again
                    }
                  }
Goal What is the sum of the first 5 prime numbers?
