# Data Attribution

## The Complete Works of William Shakespeare

- **Source**: Project Gutenberg — https://www.gutenberg.org/ebooks/100
- **Author**: William Shakespeare
- **Transcribed by**: Project Gutenberg volunteers
- **License**: Public Domain (Project Gutenberg License)
  - Full license: https://www.gutenberg.org/policy/license.html

The file `The Complete Works of William Shakespeare.txt` is used as a training
corpus for the demonstration examples in this presentation. Prebuilt data files
in `data/` (bigrams, trigrams, embeddings, tokenizer) are derived from this corpus.
